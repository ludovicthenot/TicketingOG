# TicketingOG

Base PHP pur avec PDO, sessions natives et protections backend.

## Structure

```text
config.php
db.php
auth.php
functions.php
login.php
logout.php
dashboard.php
projects.php
profile.php
assets/
tickets/
```

`config.php` reste hors Git. Le dossier `tickets/` est reserve aux pages `list.php`, `create.php` et `view.php`.
